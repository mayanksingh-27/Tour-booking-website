# *Tour-Booking-Website*
## -Run npm i
#### to install all the node modules
-----
## -Make config.env file and add following things
####    -NODE_ENV=development
####    -PORT=3000
####    -DATABASE
####    -DATABASE_PASSWORD
####    -JWT_SECRET
####    -JWT_EXPIRES_IN
####    -JWT_COOKIE_EXPIRES_IN
####    -EMAIL_USERNAME
####    -EMAIL_PASSWORD
####    -EMAIL_HOST=smtp.mailtrap.io
####    -EMAIL_PORT

-------
## -Run npm start
#### to run the website
